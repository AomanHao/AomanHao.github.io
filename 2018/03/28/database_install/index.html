4.3.1 报错

原因：用记事本配置my.ini编码格式有问题，一般情况下是UTF-8编码格式，但是这里需要ANSI编码格式
用记事本打开my.ini文件，然后点击：文件—>另存为—>将编码修改为：ANSI—>保存！

然后cmd窗口输入命令行启动mysql

4.3.2 报错
